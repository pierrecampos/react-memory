# React Memory :jigsaw:
Jogo da memória desenvolvido no curso [B7Web](https://b7web.com.br/).

## Tecnologias :gear:
* React
* TypeScript
* Styled-Components

## Para rodar o projeto :rocket:

#### Clone o repositório
```
git clone https://github.com/pierrecampos/react-memory.git
```

#### Entre na pasta
```
cd react-memory 
```

#### Instale as dependências
```
npm install
```
#### Rode o projeto!
```
npm start
```
